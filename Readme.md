NeoJetty
=========
NeoJetty is a practing project which is based on Jetty9.


how-to-build
============
mvn, Eclipse or IDEA


how-to-run
============
org.eclipse.jetty.start.Main.java


version
=========
0. Keeping all the code, remove some unnecessary file.
1. Remove test code and examples. Add IDE support.
